# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

A progress-tracking app for the Claude Code Mastery Curriculum — a 4-week self-paced learning plan. The curriculum source lives in `claude_code_lesson_plan.md` (and the original `.docx`). The app will let the user follow the lesson plan and track their completion progress.

## Curriculum Structure

The lesson plan has 4 weeks, each with multiple days and deliverables:
- **Week 1** (Days 1–7): Foundation & First Commands — installation, CLI navigation, CLAUDE.md
- **Week 2** (Days 8–14): Productive Workflows & Plan Mode — permissions, Git integration, context management
- **Week 3** (Days 15–21): Customization & Extensions — skills, hooks, MCP servers, subagents
- **Week 4** (Days 22–28): Advanced Patterns & Real-World Mastery — Agent Teams, headless mode, capstone
